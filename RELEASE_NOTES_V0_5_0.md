# Threshold Cybernetics v0.5.0 - Risk Hint Gradation Harness

v0.5.0 adds a bounded public risk-hint gradation harness that assigns non-authorizing review-priority tiers to the v0.4.1 Git-byte-verified PetClinic candidate summaries.

## What Changed

- Added a public `risk-gradation` harness.
- Added bounded rule cards for risk-hint tiers.
- Added tests for expected tiers and no-authority invariants.
- Added a public-safe PetClinic v0.5 risk gradation run.

## Boundaries

v0.5.0 depends on the v0.4.1 Git-object-byte hash provenance repair.
v0.5.0 does not claim mature risk scoring.
v0.5.0 does not claim semantic breaking-change detection.
v0.5.0 does not authorize patches, PRs, execution or deployment.

Risk hint tiers are review-priority categories only.
