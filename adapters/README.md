These adapter samples are synthetic public-safe examples.

- OpenRewrite DryRun != Patch Approval.
- SARIF Finding != Security Fix.
- GitHub Actions Pass != Deployment Readiness.

The v0.2 harness includes small parser sketches and sample payloads only. Full toolchain integration remains an evidence gap for future releases.
