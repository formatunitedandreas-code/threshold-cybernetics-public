# Public Redaction Manifest

## Removed

- Local Windows paths.
- User profile values.
- Windows account identity.
- Authentication commands and raw responses.
- Repository permission payloads.
- Active or reusable grants.
- Full human-decision artifacts.
- Full internal receipts.
- Temporary file paths.
- Raw stdout and stderr.
- Full prompt transcripts.

## Retained

- Public PR and commit references.
- Public check names and outcomes.
- Public review finding and resolution.
- Failure categories.
- Merge method.
- Merge result SHA.
- Terminal state.
- Non-claims.
- Sanitized evidence summaries.
