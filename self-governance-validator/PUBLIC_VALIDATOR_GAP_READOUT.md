# Public Validator Gap Readout

The v0.9.0 external critique pack stated that a Release Action Receipt Validator exists.

A public reproducibility review found that the validator was not available as public executable code in this repository at v0.9.0.

This is a public evidence gap.

Private validator existence is not equivalent to public reproducibility.

v0.9.1 repairs this either by publishing a minimal public validator harness or by downgrading the public claim.
