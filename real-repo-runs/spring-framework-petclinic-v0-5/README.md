# Spring Framework PetClinic Risk Hint Gradation v0.5

This directory records a bounded public-safe risk-hint gradation run over the v0.4.1 Git-object-byte verified candidate and evidence summaries.

Published artifacts contain only candidate IDs, candidate types, risk-hint categories, bounded risk-hint tiers, aggregate evidence counts, review recommendations, and non-authorizing boundaries.

No PetClinic source file contents, source snippets, scanner reports, private core files, private logs, or raw traces are published here.

## Boundary

Risk Hint Gradation != Mature Risk Scoring
Risk Tier != Semantic Breaking-Change Detection
Risk Tier != Execution Authorization
Risk Tier != Patch Approval
Risk Tier != Security Finding
Risk Tier != Deployment Readiness
