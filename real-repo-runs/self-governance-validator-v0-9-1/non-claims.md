# Non-Claims

This v0.9.1 public validator harness does not claim:

- private core publication
- mandatory git enforcement
- mandatory gh enforcement
- hook installation
- wrapper installation
- Codex runtime tool interception
- commit authorization
- tag authorization
- GitHub release authorization
- product readiness
- deployment readiness
- security remediation
- compliance certification
- runtime authority

The validator checks plan-vs-receipt consistency only.
