# Spring Framework PetClinic v0.7 Receipt Validation Run

This directory contains the public-safe v0.7.0 scoped human authorization receipt validation run.

Inputs:

- `../spring-framework-petclinic-v0-6/policy-mapping.public-safe.json`
- synthetic receipts from `../../authorization-receipts/receipts/`

The run checks whether synthetic receipt scopes match required policy-lane review scopes. It does not grant authorization, execute patches, mutate source, create PRs, merge, deploy, claim security remediation, claim product readiness, claim mature risk scoring, claim semantic breaking-change detection, publish real reviewer data, or grant runtime authority.
