# Known Limitations

- Release Action Receipt Validator v0.1 exists but is not mandatory enforcement.
- There is no mandatory git wrapper.
- There is no mandatory gh wrapper.
- There is no mandatory pre-commit, pre-push, pre-tag, or release hook.
- Threshold does not claim observed Codex runtime tool interception.
- Boundary soundness checks are public harness checks, not certification.
- Public evidence remains public-safe summary material, not private core validation.
- The project does not claim product readiness, deployment readiness, security remediation, compliance certification, autonomous refactoring, or runtime authority.
