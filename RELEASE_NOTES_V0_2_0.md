# Threshold Cybernetics v0.2.0 - Public Evidence Harness

v0.2.0 moves the public release from proof-of-architecture toward proof-of-evidence by adding a minimal reproducible harness, sample decision envelopes, run logs, CI checks and a red-team evidence-gap register.

## Added

- Public synthetic candidate ledger and decision envelopes.
- Minimal TypeScript authorization-gate harness.
- Stop/No-Action fixtures for DEP-001, CFG-001, and JSP-001.
- Local run logs and produced decision envelopes.
- GitHub Actions authorization-gate workflow.
- Tool adapter samples for OpenRewrite, GitHub Actions, and SARIF.
- Red-team critique, response, accepted criticism, and evidence-gap register.

This release does not claim product readiness.
This release does not claim deployment readiness.
This release does not claim security remediation.
