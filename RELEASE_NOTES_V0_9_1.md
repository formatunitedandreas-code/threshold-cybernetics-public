# Threshold Cybernetics v0.9.1 - Public Release Action Receipt Validator Harness

v0.9.1 repairs the public evidence gap in v0.9.0 by adding a reproducible public Release Action Receipt Validator harness with schemas, fixtures, tests and adversarial cases.

## What Changed

- Added a public-safe minimal validator harness under `self-governance-validator/`.
- Added plan and receipt schemas.
- Added valid and adversarial public-safe fixtures.
- Added tests for release tag, allowed actions, file allowlist, forbidden actions and forbidden claims.
- Added public run artifacts under `real-repo-runs/self-governance-validator-v0-9-1/`.
- Updated the external critique pack to distinguish private-reported validator status from public reproducibility before v0.9.1.

## Non-Claims

This release does not make the validator mandatory for git/gh.
This release does not install hooks.
This release does not install wrappers.
This release does not claim Codex runtime tool enforcement.
This release does not claim product readiness, deployment readiness, security remediation, compliance certification or runtime authority.
