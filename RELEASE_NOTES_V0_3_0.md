# Threshold Cybernetics v0.3.0 - Public Candidate Detection Evidence Harness

v0.3.0 adds a minimal public candidate-detection harness that derives DEP-001, CFG-001 and JSP-001 from synthetic open fixture surfaces, binds evidence hashes and preserves the boundary that candidate detection is not patch approval.

## Added

- Synthetic Mini Spring Legacy fixture surfaces.
- Candidate detection rule cards for dependency/plugin, Spring config, and JSP/MVC surfaces.
- Deterministic TypeScript candidate detection harness.
- SHA-256 evidence binding for detected fixture files.
- Candidate detection tests and no-execution invariant tests.
- Candidate detection run logs and produced candidate ledgers.
- GitHub Actions candidate-detection workflow.
- Updated red-team response and evidence-gap register.
