# Boundary-Soundness Public Demo Boundary v0.1

Threshold Cybernetics Public includes a bounded Boundary-Soundness readout that checks whether forbidden authority promotions are reachable across the public PetClinic candidate -> evidence -> risk -> policy -> receipt path.

The v0.8.0 harness models public governance states and verifies that candidate, evidence, risk, policy and receipt states do not reach patch, PR, merge, deployment or runtime authority markings within the bounded demo.

This is not a claim that Threshold formally proves all workflows sound. It is not a BPMN or DMN engine. It does not enforce runtime safety, prove compliance, guarantee safe refactoring, claim production readiness, or grant action authority.
